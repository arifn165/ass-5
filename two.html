Question-1: What are the different ways to select an element in the DOM?

Answer No 1 :- DOM থেকে এলিমেন্ট সিলেক্ট করার উপায়: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()


What is the difference between innerHTML, innerText, and textContent ?

Answer No 2 :- innerHTML : HTML সহ পুরো কনটেন্ট নেয়, innerText : ভিজিবল টেক্সট নেয়, আর textContent : সব টেক্সট নেয় (হিডেনসহ)।


What is event delegation in the DOM?

Answer No 3 :- Event Delegation হল একটি parent element-এ event listener যোগ করা, যাতে child elements-এ event ঘটলে সেটা handle করা যায়।


What is event bubbling in the DOM?

Answer No 4 : Event Bubbling হল যখন child element-এ event ঘটে, তখন তা ক্রমশ তার parent elements গুলোতে propagate হয়।


How do you create, add, and remove elements using JavaScript?

Answer No 5 :- JavaScript দিয়ে এলিমেন্ট তৈরি করতে createElement(), যোগ করতে appendChild(), এবং মুছে ফেলতে removeChild() ব্যবহার করা হয়।
